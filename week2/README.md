The House Prices Regression problem 

The dataset obtained from Kaggle: https://www.kaggle.com/datasets/yasserh/housing-prices-dataset
The used models: LinearRegression, Lasso, RandomForest, XGBoost
The best model - LinearRegression 

The following changes to dataset has been made: EDA, encoding, scaling, feature engineering, log-transformation, train-test split, cross-validation, 
comparable performance metrics, feature importance, conclusion.
